基础用法

注意: 如果你想让属性data-attribute的值为true的话，你必须删除该属性，如果仅仅将该属性的值修改为false的话,比如， data-widget-togglebutton="true"，它仍将显示为true的效果。
HTML5 data 属性 / CSS / HTML 追加都后面的属性值 示例 描述
data-widget-sortable header false 小部件不做排序(只有设置为false时才起作用).
data-widget-icon .jarviswidget fa fa-trash 在部件的头部使用一个图标(要查看所有的图表，请浏览图表章节).
data-widget-togglebutton .jarviswidget false 防止部件出现一个切换按钮(只有设置为false时才起作用).
data-widget-deletebutton .jarviswidget false 防止部件出现一个删除按钮(只有设置为false时才起作用).
data-widget-editbutton .jarviswidget false 防止部件出现一个编辑按钮(只有设置为false时才起作用).
data-widget-fullscreenbutton .jarviswidget false 防止部件出现一个全屏按钮(只有设置为false时才起作用).
data-widget-load .jarviswidget widget-ajax.php 通过ajax加载的文件。
data-widget-refresh .jarviswidget 10 快速刷新一个ajax文件(请查看 'data-widget-load').
data-widget-refreshbutton .jarviswidget false 显示刷新按钮时，防止出现一个ajax小部件(只有设置为false时才起作用).
data-widget-hidden .jarviswidget true 在加载时，隐藏该部件(只有设置为true时才起作用).
data-widget-colorbutton .jarviswidget false 默认是激活状态，设置为false时，隐藏。
data-widget-collapsed .jarviswidget false 加载时，小部件是折叠的。(只有设置为true时才起作用). 当加载任何页面时，总是处于折叠状态。
data-widget-grid article false 你可以将可排序或者可删除从这个网格中排除，意思是在这个区域的所有widget都可以正常工作，但不能删除，也不能做排序。 将该属性(只有设置为false时才起作用)添加到grid元素中。
.no-padding .widget-body n/a 移除所有的内部控件填充体
.well .jarviswidget n/a 将一个控件转化为一个容器
这些HTML5属性可以在widget选项中单独使用。 主要选项可以在widgets插件中自动改变。 注意: data属性只能有一个boolean值(请参照上面)!
